# Teste Prático para Desenvolvedor Full Stack Kronoos

por.: [Átila de Freitas](https://www.linkedin.com/in/atilafreitas/)

para rodar corretamente o projeto:

### Instação
```bash
npm install
```

### Para rodar o projeto
```bash
npm run build
```

### Para iniciar o projeto
```bash
npm start
```

### Para rodar o teste
```bash
npm test
```
